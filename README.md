## 📌 Project Overview

An end-to-end sales data analysis of **Vrinda Store** for the year **2022**, built entirely using **Microsoft Excel**.  
The project includes data cleaning, pivot table analysis, chart creation, and an interactive dashboard with slicers.

---

## 🎯 Objective

To analyze Vrinda Store's annual sales data and answer key business questions:

- Which **gender** buys more?
- Which **states** contribute most to revenue?
- Which **age group** places the most orders?
- Which **sales channels** perform best?
- What is the **monthly sales trend**?
- What is the **order delivery success rate**?

---

## 📊 Key Insights

| 📍 Insight | 🔍 Finding |
|---|---|
| 👩 Top Buyers | Women (~64% of total sales) |
| 🏆 Best State | Maharashtra (₹29.9L) |
| 🛒 Best Channel | Amazon (35% of orders) |
| 📦 Delivery Rate | 92%+ successfully delivered |
| 👩‍🦱 Top Segment | Adult Women (30–50 age group) |
| 📅 Peak Month | March (highest sales & orders) |
| 💰 Total Revenue | ₹2.12 Crore (approx) |

## 📁 File Structure

```
Vrinda_Store_Data_Analysis.xlsx
├── Vrinda Store          → Raw Data (31,000+ rows)
├── Sales VS Order        → Monthly Sales & Order Trend
├── Men VS Women          → Gender-wise Revenue Split
├── Order Status          → Delivered, Cancelled, Returned
├── Sales Top 5 State     → State-wise Revenue
├── Age & Gender          → Age Group × Gender Orders
├── Sheet7                → Channel-wise Order Share
└── Vrinda Store Report   → 📊 Final Interactive Dashboard
```
